# Contribuindo com o Projeto

Obrigado por considerar contribuir com este projeto! Aqui estão algumas diretrizes para ajudar você:

## Como Contribuir

1. **Fork o Repositório**: Clique no botão "Fork" no topo da página do repositório.
2. **Clone o seu Fork**: Clone o repositório forkado para o seu ambiente local.
3. **Crie uma Branch**: Crie uma nova branch para a sua contribuição.
4. **Faça as Alterações**: Implemente as mudanças ou adições que deseja.
5. **Teste as Alterações**: Certifique-se de que tudo está funcionando como esperado.
6. **Envie um Pull Request**: Submeta as suas alterações para revisão.

## Diretrizes de Codificação

- Utilize nomes de variáveis e funções descritivos.
- Adicione comentários explicativos onde necessário.
- Mantenha a consistência no estilo de codificação.

## Relatando Problemas

Se encontrar um bug ou tiver uma sugestão, por favor, abra uma issue no repositório.

Agradecemos a sua contribuição!
