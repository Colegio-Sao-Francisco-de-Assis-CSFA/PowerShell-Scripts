# Scripts da categoria `dhcp`

Scripts relacionados a consultas e alterações em infraestrutura DHCP.

## Dependências comuns

- permissões administrativas no ambiente Windows
- módulos e cmdlets de DHCP disponíveis na máquina

## Status

Pasta ainda aguardando revisão completa para o padrão público.

## Scripts

- `add-dhcp-reservation.ps1` — Adiciona reservas DHCP com base em um CSV.
- `get-dhcpleases.ps1` — Consulta leases ativos de um escopo DHCP.
- `set-dhcpremote-machine.ps1` — Executa ações remotas relacionadas a DHCP e rede.
