# Scripts da categoria `user`

Lista de scripts disponíveis nesta categoria:

- `change-password.ps1` — Altera a senha de um usuário do Active Directory.
- `create-folders.ps1` — Cria pastas automaticamente a partir de uma lista.
- `create-new-funcionarios.ps1` — Cria contas de funcionários no domínio.
- `create-new-students.ps1` — Cria contas de alunos no domínio.
- `create-single-user.ps1` — Cria uma conta de usuário individual.
- `create-user-alias.ps1` — Adiciona um alias de e-mail para o usuário.
- `delete-files.ps1` — Exclui arquivos com base em critérios definidos.
- `move-items-create-folders.ps1` — Move arquivos para pastas com base em um CSV.
- `suspend-single-user.ps1` — Suspende uma conta de usuário.
- `suspend-users.ps1` — Suspende múltiplos usuários de uma vez.
- `unsuspend-users-2.0.ps1` — Versão melhorada do script de reativação de usuários.
- `unsuspend-users.ps1` — Reativa usuários suspensos.
- `update-dhcp.ps1` — Atualiza configurações do servidor DHCP.
- `update-display-name.ps1` — Atualiza o nome de exibição dos usuários.
- `update-func-photos.ps1` — Atualiza fotos de funcionários.
- `update-student-photos.ps1` — Atualiza fotos de estudantes.
- `update-user-emails.ps1` — Atualiza os e-mails principais dos usuários.
- `update-user-ou.ps1` — Move os usuários para uma nova OU no Active Directory.
- `update-user-photos.ps1` — Atualiza fotos dos usuários no domínio.
