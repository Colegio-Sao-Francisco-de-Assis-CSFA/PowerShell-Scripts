# Scripts da categoria `classroom`

Scripts voltados para automações no Google Classroom, em geral usando GAM.

## Dependências comuns

- GAM
- permissões adequadas no Google Workspace

## Status

Esta pasta está em revisão para adoção completa do padrão público.
Alguns scripts ainda precisam de sanitização de caminhos, validações e documentação mais detalhada.

## Scripts

- `classroom-manager-add-teacher.ps1` — Adiciona professores em turmas do Classroom.
- `classroom-manager-adms.ps1` — Gerencia administradores ou responsáveis por turmas.
- `classroom-manager-arquivar.ps1` — Arquiva turmas do Classroom em lote.
- `classroom-manager-criar.ps1` — Cria turmas no Classroom com base em dados externos.
- `classroom-manager-groups.ps1` — Gerencia grupos auxiliares usados em fluxos do Classroom.
- `classroom-manager-sync-students.ps1` — Sincroniza estudantes entre turmas e grupos.
- `classroom-manager-sync-teachers.ps1` — Sincroniza professores entre turmas e grupos.
- `classroom-manager-transfer-ownership.ps1` — Transfere a propriedade de turmas.
- `old-classroom-manager-transfer-ownership.ps1` — Versão antiga mantida para referência.
