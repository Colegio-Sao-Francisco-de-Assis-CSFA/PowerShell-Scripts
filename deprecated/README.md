# Scripts Descontinuados

Este diretório contém scripts que foram **descontinuados**, substituídos por versões mais recentes ou que não fazem mais parte do fluxo atual de trabalho.

Eles são mantidos aqui apenas para referência ou histórico, e **não devem ser usados em novos projetos**.

### Motivos comuns para descontinuação:

- Melhorias de performance ou legibilidade em novos scripts.
- Mudança de ferramentas ou processos.
- Centralização ou unificação de funcionalidades.

---

**Importante:**  
Se você está procurando a versão mais atualizada de um script, verifique a pasta principal ou consulte a documentação do projeto.
