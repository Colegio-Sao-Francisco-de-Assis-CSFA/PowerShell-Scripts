# Scripts da categoria `drive`

Scripts voltados para envio e organização de arquivos no Google Drive.

## Dependências comuns

- GAM
- acesso adequado às contas e pastas do Google Drive

## Status

Pasta em revisão.
Parte dos scripts já está sendo ajustada para o padrão público e parte ainda depende de revisão adicional.

## Scripts

- `upload-arquivos-drive.ps1` — Envia arquivos de uma pasta local para uma pasta no Google Drive.
- `upload-audios-to-drive.ps1` — Envia áudios personalizados para o Google Drive dos usuários.
