# Scripts da categoria `group`

Scripts para criação, limpeza e sincronização de grupos, principalmente no Google Workspace via GAM.

## Dependências comuns

- GAM
- permissões para administrar grupos

## Status

Pasta em revisão para padronização pública.

## Scripts

- `add-studentsto-groups.ps1` — Adiciona alunos a grupos específicos via GAM.
- `gam-limpar-membros-grupos.ps1` — Limpa membros de grupos usando um grupo de referência vazio.
- `sync-group-members.ps1` — Sincroniza membros entre grupos com base em um arquivo CSV.
